Automatically generates Colorbox rel tags based on post id, in turn creating automatic image grouping.

== Description ==

This is a plugin for the Lightbox clone Colorbox http://colorpowered.com/colorbox/ Visit the website to learn how to customize it. 

Jquery Colorbox is included in this plugin, the 5 premade styles that come prepackaged with the Colorbox script, and a "no overlay" style.

**Simple Cbox** automates the grouping process with simple lightweight jquery.  Enable the Global Colorbox, and the plugin will insert rel tags, and jquery variables based on the Post Id, in turn grouping each posts images in a popup colorbox slideshow.

It should be known that this plugin should be used by those that have some code interpretation skills, though coding skills are not required, they do come in handy when wanting to customize this plugin to match specific web design needs.

== Installation ==

1. Upload folder `/simple_cbox/` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings >> Simple Cbox to customize appearance and settings.
4. Enable "Global Colorboxing"
